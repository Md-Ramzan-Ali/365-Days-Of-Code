# Shortest Completing Word (LeetCode 748)

একটি জাভা (Java) ভিত্তিক সমাধান যা নির্দিষ্ট একটি লাইসেন্স প্লেটের (`licensePlate`) সমস্ত ক্যারেক্টার ধারণকারী শব্দগুলোর মধ্যে সবচেয়ে ছোট এবং প্রথম শব্দটিকে খুঁজে বের করে।

## সমস্যা বিবরণী (Problem Description)

একটি স্ট্রিং `licensePlate` এবং একটি স্ট্রিং অ্যারে `words` দেওয়া আছে। আমাদের সবচেয়ে ছোট **Completing Word**-টি খুঁজে বের করতে হবে।

একটি শব্দকে তখন মেলা বলা হবে যখন:
- `licensePlate`-এর সমস্ত অক্ষর (case-insensitive) শব্দটিতে উপস্থিত থাকবে।
- লাইসেন্স প্লেটের সংখ্যা এবং স্পেস উপেক্ষা করা হবে।
- যদি কোনো অক্ষর একাধিকবার থাকে, তবে শব্দটিতেও সেটি অন্তত ততবার বা তার বেশি থাকতে হবে।

যদি একাধিক ছোট শব্দ পাওয়া যায়, তবে `words` অ্যারেতে যেটি প্রথমে এসেছে, সেটি রিটার্ন করতে হবে।

### উদাহরণ ১:
```text
Input: licensePlate = "1s3 PSt", words = ["step","steps","stripe","stepple"]
Output: "steps"
Explanation: 's', 'p', 's', 't' ধারণকারী সবচেয়ে ছোট শব্দ হলো "steps"।
