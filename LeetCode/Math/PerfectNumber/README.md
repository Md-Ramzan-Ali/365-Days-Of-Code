# Perfect Number (LeetCode 507)

A highly optimized, $O(1)$ time complexity Java solution to check whether a given integer is a perfect number.

## Problem Description

A **perfect number** is a positive integer that is equal to the sum of its positive divisors, excluding the number itself. Given an integer `num`, return `true` if it is a perfect number, otherwise return `false`.

### Example 1:
```text
Input: num = 28
Output: true
Explanation: 28 = 1 + 2 + 4 + 7 + 14
Only 1, 2, 4, 7, and 14 are all positive divisors of 28.
